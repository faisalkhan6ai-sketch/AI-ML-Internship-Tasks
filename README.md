Task 1: Exploring and Visualizing a Simple Dataset
Perform exploratory data analysis on the Iris dataset to understand
the structure, distribution, and relationships between features
using Python libraries.
Dataset Used:
Name:Iris Dataset
Source:iris.csv
Size:150 rows, 6 columns
Features:SepalLengthCm, SepalWidthCm, PetalLengthCm, PetalWidthCm, Species
Tools & Libraries Applied:
1) Python
2) Pandas — data loading and summary statistics
3) Matplotlib — basic plotting
4) Seaborn — scatter plots, histograms, box plots
Key Results and Findings:
Scatter Plot
- Petal features separate the 3 species very clearly
- Sepal features overlap more between species
- Iris Setosa is always clearly separate from others
Histograms
- Petal Length and Petal Width show bimodal distribution
- Setosa has very small petal size compared to others
Box Plots
- SepalWidthCm has the most outliers
- Iris Virginica has the largest petal dimensions
- Iris Setosa has the smallest and most consistent petals
Conclusion
Petal features (Length and Width) are the most useful for
identifying and separating Iris species. Sepal features
alone are not reliable enough for classification.
